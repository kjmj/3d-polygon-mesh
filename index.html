KeyMappings

x - Translate the model in the positive x direction
c - Translate the model in the negative x direction
y - Translate the model in the positive y direction
u - Translate the model in the negative y direction
z - Translate the model in the positive z direction
a - Translate the model in the negative z direction
r - Rotate the model in an X-roll about it's current position
b - Toggle pulsing meshes ON/OFF. When ON, the mesh faces pulse back and forth continuously
